# Parsec-lite: سیستم استریم بازی با عملکرد بالا در شبکه محلی (LAN)

این پروژه یک سیستم استریم بازی با تاخیر بسیار کم است. این پروژه شامل دو نسخه است: نسخه اصلی با زبان C++ برای عملکرد نهایی و یک نسخه پیش‌نمایش (Prototype) با زبان پایتون.

## 🚀 شروع سریع

### ۱. نسخه اصلی (C++) - پیشنهادی برای ویندوز
این نسخه بهترین عملکرد و کمترین تاخیر را دارد.

#### نیازمندی‌ها
- ویندوز ۱۰ یا ۱۱
- کارت گرافیک NVIDIA، AMD یا Intel با قابلیت انکود سخت‌افزاری
- درایور ViGEmBus (برای پشتیبانی از کنترلر)

#### نحوه بیلد (Build)
۱. ابتدا یک پوشه برای بیلد بسازید:
```bash
mkdir build
cd build
```
۲. با استفاده از CMake پروژه را پیکربندی کنید:
```bash
cmake ..
```
۳. پروژه را کامپایل کنید:
```bash
cmake --build . --config Release
```

#### نحوه اجرا
- **اجرای سرور (Host):**
  ```bash
  ./parsec-lite.exe --host
  ```
- **اجرای کلاینت (Client):**
  ```bash
  ./parsec-lite.exe --client <IP_SERVER>
  ```

---

### ۲. نسخه پایتون (Prototype) - برای تست سریع و آموزش
این نسخه ساده‌تر است و روی لینوکس و مک نیز قابل اجراست (البته با قابلیت‌های محدودتر).

#### نصب نیازمندی‌ها
```bash
pip install -r requirements.txt
```

#### نحوه اجرا
- **اجرای سرور (Host):**
  ```bash
  python3 host/streamer.py
  ```
- **اجرای کلاینت (Client):**
  ```bash
  python3 client/client.py <IP_SERVER>
  ```
