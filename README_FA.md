# Parsec-lite: سیستم استریم بازی با عملکرد بالا در شبکه محلی (LAN)

این پروژه یک سیستم استریم بازی با تاخیر بسیار کم است که با زبان C++ برای عملکرد نهایی توسعه یافته است.

## 🚀 شروع سریع

### نیازمندی‌ها
- ویندوز ۱۰ یا ۱۱
- کارت گرافیک NVIDIA، AMD یا Intel با قابلیت انکود سخت‌افزاری
- درایور ViGEmBus (برای پشتیبانی از کنترلر)

### نحوه بیلد (Build)
۱. ابتدا یک پوشه برای بیلد بسازید:
```bash
mkdir build
cd build
```
۲. با استفاده از CMake پروژه را پیکربندی کنید:
```bash
cmake ..
```
۳. پروژه را کامپایل کنید:
```bash
cmake --build . --config Release
```

### نحوه اجرا
- **اجرای سرور (Host):**
  ```bash
  ./parsec-lite.exe --host
  ```
- **اجرای کلاینت (Client):**
  ```bash
  ./parsec-lite.exe --client <IP_SERVER>
  ```
